FAQ

Common Questions

Does Azure Builder have access to my Azure account?

Only when you explicitly connect it. We use OAuth with minimal permissions. You can review and approve every command before it runs — nothing executes without your confirmation.

What if the AI generates incorrect commands?

Every generated command goes through validation before execution. You always get a review step. Our AI is trained on Azure's official documentation and continuously improved with user feedback.

Can I use Azure Builder with existing infrastructure?

Absolutely. Azure Builder can scan your existing resource groups and understand your current setup. It generates commands that work alongside what you already have deployed.

Is my data secure?

We never store your Azure credentials. All communication is encrypted end-to-end. We're SOC 2 Type II compliant, and our infrastructure runs on Azure itself with enterprise-grade security.

Can I export the generated commands?

Yes! Export as shell scripts, ARM templates, Bicep files, or Terraform configurations. Use Azure Builder to generate, then integrate into your existing CI/CD pipeline.

Do you support Azure Government and sovereign clouds?

Enterprise plans include support for Azure Government, Azure China, and Azure Germany. Contact our sales team for sovereign cloud configurations.
