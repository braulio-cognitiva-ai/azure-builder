FAQ

Common Questions

How does pricing estimation work?

We use the Azure Retail Prices API to fetch real-time pricing for every resource in your proposed architecture. Costs are calculated based on the region, SKU, and estimated usage patterns you specify. Prices update automatically as Azure's pricing changes.

Is it safe to connect my Azure subscription?

Absolutely. We use OAuth2 authentication with minimal permissions. We never store your credentials permanently—everything is handled through secure token exchange. You can revoke access at any time from your Azure portal. All deployments require your explicit approval.

Can I modify the generated architecture before deploying?

Yes! Every architecture proposal is fully customizable. You can adjust SKUs, regions, configurations, and individual resources before deployment. Think of AI proposals as a starting point—you have complete control over the final infrastructure.

What if a deployment fails?

Deployments use Azure Resource Manager's transactional guarantees. If any part of the deployment fails, the entire stack rolls back automatically. You'll get detailed error logs, and our support team can help diagnose issues on Pro and Enterprise plans.

Do I need Azure expertise to use this?

Not at all. That's the whole point. You describe what you need in plain English, and Azure Builder handles the architecture, resource selection, and configuration. Beginners can deploy production-grade infrastructure in minutes. Experts save time by skipping the manual design work.

What Azure services are supported?

We support 80+ Azure services including App Service, AKS, Functions, Cosmos DB, SQL Database, PostgreSQL, Storage, Networking, AI/ML services, and more. If a service isn't supported yet, let us know—we prioritize based on customer requests.

Can I use existing resources in my subscription?

Yes! Azure Builder scans your existing infrastructure and proposes architectures that integrate with what you already have. It won't duplicate resources unnecessarily and will reuse existing VNets, storage accounts, and other shared infrastructure when appropriate.

What's included in the audit trail?

Every architecture proposal, deployment, configuration change, and user action is logged with timestamps and user attribution. Pro and Enterprise plans include full audit exports for compliance reviews. Enterprise plans get customizable compliance reports for SOC 2, HIPAA, and other frameworks.
